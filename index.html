Projects

Things I've
made.

01
SmartBin

Arduino-powered Smart Bin that uses an ultrasonic sensor and servo motor to automate lid operation, solving the hygiene issues associated with touching dirty surfaces. By detecting a user's proximity for hands-free disposal, this project taught me how to effectively bridge the gap between digital sensor data and physical mechanical motion.

C++ Robotics
02
MIRA (Multi Intelligent Robot Assistant)

An autonomous robotic system powered by a Raspberry Pi 5 and Ollama to run local LLMs. This project addresses the need for a privacy-centric, offline AI assistant that can interact with the physical world via a mobile chassis. Through this ongoing build, I am learning to optimize edge computing performance and bridge the gap between high-level AI reasoning and real-time hardware execution.

Research Robotics
03
Connect4

I built a Python-based Connect 4 game that allows players to compete digitally, replacing the need for a physical board and pieces. This project solves the limitation of physical proximity by providing a portable and automated gaming environment with instant win-condition logic. Through this development, I mastered managing 2D arrays to represent the game grid and implemented efficient algorithms to detect vertical, horizontal, and diagonal victory patterns.

Python Game
04
SYNAPSE

I am developing SYNAPSE, a smart electrostatic air purifier that integrates AI to predict dust accumulation (fouling) and optimize an automatic cleaning system. This project solves the problem of declining filtration efficiency in high-density urban areas by ensuring the device maintains peak performance without relying on manual maintenance. Through this research, I am mastering the use of predictive algorithms to bridge the gap between environmental data and autonomous hardware maintenance.
